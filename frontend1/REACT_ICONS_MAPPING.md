## Usage Example

```jsx
// Import icons
import { MdDelete, MdEdit, MdAdd, MdKeyboardArrowDown } from 'react-icons/md';

// Use in component
<button>
  <MdDelete className="w-5 h-5" />
</button>
```

## Size Classes

- Small: `className="w-4 h-4"`
- Medium: `className="w-5 h-5"`
- Large: `className="w-6 h-6"`
